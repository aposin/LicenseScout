licensescout-report:report

Note:This goal should be used as a Maven report.

Full name:

org.aposin.licensescout:licensescout-report-maven-plugin:1.3.1-SNAPSHOT:report

Description:

Scans Java installation directory for licenses and generates a Maven site report.

Attributes:

  • Requires a Maven project to be executed.
  • Executes only as a reportSet (reporting goal).
  • Requires dependency resolution of artifacts in scope: runtime.
  • Binds by default to the lifecycle phase: site.

Required Parameters

Name Type Since Description
<scanDirectory> File 1.1 Directory to scan for archives.

This is the directory where the LicenseScout will start to traverse directories recursively. If the directory does not exist, the LicenseScout will be terminated with an exception.


User property is: scanDirectory.
